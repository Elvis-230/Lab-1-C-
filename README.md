# Lab Week 1: Prefix Sums

## Overview
Implements two functions to check prefix sums of arrays with +1 and -1 values:
- `prefixNonNegative()` - checks if all prefix sums stay >= 0
- `prefixNonPositive()` - checks if all prefix sums stay <= 0

## Build
```bash
g++ -o build/prefix_check src/prefix_check.cpp
```

## Run
```bash
./build/prefix_check
```

This will run all unit tests and display results.
